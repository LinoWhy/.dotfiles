输出格式：

1. 总体判断：patch 是否正确/是否存在风险，用 1–2 句说明原因。
2. 问题列表（按优先级从高到低；若无问题请写“未发现导致回退或正确性问题”，并简述已检查的风险点）：
   - [P{0-3}] {简短标题} — {file_path}:{start_line}-{end_line}
     {问题原因/触发条件/影响 + 明确的修复建议}
